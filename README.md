# Smart-Blood-Bank-Donor-Management-System
This is a healthcare application that uses ShaktiDB to manage donor records, blood inventory, donation history, and hospital blood requests. The system helps track blood availability, process requests efficiently, and generate reports on donation and demand trends, ensuring better management and utilization of blood resources.
